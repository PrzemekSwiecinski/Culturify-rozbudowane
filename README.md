# Rozbudowana aplikacja Culturify / The extensive Culturify app

(PL) Rozbudowana aplikacja Culturify o dodatkowych widokach oraz funkcjonalnościach takich jak:
- zmiana motywu strony;
- rozbudowana galeria z możliwością dodawania zdjęć przez admininstratora oraz możliwością zmiany kolejności wyświetlanych zdjęć;
- blog z postami akceptowanymi przez administratora;
- dodany moderator (oraz jego logowanie);
- możliwość zmiany kategorii dla moderatora przez administratora.

Data stworzenia projektu: 01.01.2025 - 04.02.2025.

(EN) An expanded Culturify application with additional views and functionalities, including:
- changing the page theme;
- an expanded gallery with the ability for the administrator to add photos and change the order of displayed photos;
- a blog with posts approved by the administrator;
- added moderator (and their login);
- the ability for the administrator to change the moderator's category.

Project creation date: 01.01.2025 - 04.02.2025.
